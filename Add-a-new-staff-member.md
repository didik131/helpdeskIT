How to add a new staff member
=======

## Add a new staff member ##
 1. Go to "Staff"
 2. Go to "Staff Members"
 3. Click on "Add new staff" button
 4. Input some Name
 5. Input some Email
 6. Input some Password
 7. Choose one Level
 8. Choose one or more Departments
 9. Click on "SAVE" button
